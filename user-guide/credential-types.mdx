---

## Referencia

Cada credencial en Tu Pana tiene un `credential_type_id` que indica el tipo de servicio o plataforma a la que corresponde. Este valor se usa al crear credenciales con `POST /credentials`.

## Tipos disponibles

| ID | Nombre | Descripcion |
|----|--------|-------------|
| 70 | `SII` | Servicio de Impuestos Internos (clave tributaria) |
| 73 | `sii_company` | SII a nivel de empresa (clave de empresa) |
| 75 | `certificate` | Certificado digital (.pfx) |
| 84 | `bank_chile` | Banco de Chile |
| 85 | `bank_estado` | BancoEstado |
| 86 | `bank_santander` | Santander |
| 87 | `bank_scotiabank` | Scotiabank |

<Note>
  Los IDs son estables y no cambian. Usa el `credential_type_id` numerico al crear credenciales via API.
</Note>

## Ejemplo de uso

Para crear una credencial SII:

```json
{
  "master_entity_id": 123,
  "credential_type_id": 70,
  "user_rut": "12345678-9",
  "password": "mi-clave-sii"
}
```

Para subir un certificado digital:

```json
{
  "master_entity_id": 123,
  "credential_type_id": 75,
  "digital_certificate_password": "clave-del-pfx",
  "digital_certificate_base64": "MIIK..."
}
```
