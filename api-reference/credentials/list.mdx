---
title: "Listar Credenciales"
openapi: "GET /credentials"
---

## Qué hace

- Devuelve todas las credenciales activas del usuario autenticado
- Incluye el estado de validación de cada credencial
- Muestra la fecha de creación de cada credencial
- Lista las entidades maestras asociadas a cada credencial

## Ejemplos de uso

- Verificar qué credenciales están configuradas
- Consultar el estado de validación de las credenciales
- Listar todas las credenciales antes de crear una nueva
- Revisar qué entidades tienen credenciales asociadas

### Estados de Credenciales

- **VALID**: Credencial válida y activa en el SII
- **INVALID**: Credencial con datos incorrectos
- **EXPIRED**: Credencial que ha expirado
