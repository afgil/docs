---
title: 'Obtener documento programado'
openapi: 'GET /scheduled-documents/{id}'
---

## Qué hace

- Obtiene los detalles completos de un documento programado específico
- Muestra toda la información de configuración del documento programado
- Incluye información del emisor, receptor, frecuencia, próxima ejecución
- Proporciona el estado actual y el número de ejecuciones completadas
- Muestra todos los detalles de productos/servicios del documento

## Ejemplos de uso

- Consultar los detalles de un documento programado específico
- Verificar la próxima fecha de ejecución de un documento
- Revisar la configuración de un documento programado antes de actualizarlo
- Consultar cuántas veces se ha ejecutado un documento programado
- Verificar el estado actual (activo, inactivo, completado)
