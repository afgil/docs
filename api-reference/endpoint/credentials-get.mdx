---
title: 'Listar Credenciales'
api: 'GET /api/v1/credentials'
description: 'Obtiene todas las credenciales activas del usuario autenticado'
---

Este endpoint devuelve una lista de todas las credenciales activas asociadas a las entidades maestras a las que el usuario tiene acceso.

### Autenticación

<Note>
Este endpoint requiere autenticación mediante API Key.
</Note>

### Response

<ResponseField type="array">
  Array de objetos de credenciales con los siguientes campos:

<Expandable title="Credencial">
  <ResponseField name="id" type="integer">
    ID único de la credencial
  </ResponseField>

  <ResponseField name="user" type="string">
    RUT del usuario de la credencial
  </ResponseField>

  <ResponseField name="master_entity" type="object">
    <Expandable title="master_entity">
      <ResponseField name="id" type="integer">
        ID de la entidad maestra
      </ResponseField>
      <ResponseField name="name" type="string">
        Nombre de la entidad maestra
      </ResponseField>
      <ResponseField name="tax_id" type="string">
        RUT de la entidad maestra
      </ResponseField>
    </Expandable>
  </ResponseField>

  <ResponseField name="credential_type" type="object">
    <Expandable title="credential_type">
      <ResponseField name="id" type="integer">
        ID del tipo de credencial
      </ResponseField>
      <ResponseField name="name" type="string">
        Nombre del tipo de credencial
      </ResponseField>
    </Expandable>
  </ResponseField>

  <ResponseField name="status" type="string">
    Estado de la credencial (VALID, INVALID, EXPIRED)
  </ResponseField>

  <ResponseField name="created_at" type="string">
    Fecha de creación de la credencial en formato ISO 8601
  </ResponseField>
</Expandable>
</ResponseField>

### Ejemplos

<CodeGroup>
```bash Request
curl https://api.tupana.com/api/v1/credentials \
  -H "Authorization: Api-Key YOUR-API-KEY"
```

```json Response
[
  {
    "id": 1,
    "user": "12345678-9",
    "master_entity": {
      "id": 123,
      "name": "Empresa Ejemplo SpA",
      "tax_id": "76543210-1"
    },
    "credential_type": {
      "id": 70,
      "name": "SII"
    },
    "status": "VALID",
    "created_at": "2024-01-01T10:00:00Z"
  }
]
```
</CodeGroup>
