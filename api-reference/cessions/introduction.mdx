## Para que sirve la API de Cesiones?

La **API de Cesiones** permite consultar y gestionar las **cesiones de facturas** (AEC - Archivo Electronico de Cesion) desde tu integracion. Una cesion es la transferencia del derecho de credito de una factura a un tercero (por ejemplo una empresa de factoring).

Con esta API puedes:

- **Listar** todas las cesiones emitidas por una entidad (con filtros por fecha, estado, busqueda).
- **Obtener el detalle** de una cesion concreta, incluidos eventos de trazabilidad en el SII.
- **Obtener la URL del certificado de cesion** (PDF) del SII via `certificate_pdf_url` en el detalle.
- **Crear cesiones en lote** (generar AECs y enviarlos al SII) desde el endpoint de batch.

## Flujo tipico

1. **Listar cesiones** con `GET /cessions?master_entity_id={id}` para ver todas las cesiones de la entidad.
2. **Obtener una cesion** con `GET /cessions/{id}` para ver detalle, eventos y `certificate_pdf_url`.
3. **Crear nuevas cesiones** con `POST /cessions/batch` enviando los IDs de documentos y datos del cesionario.

## Autenticacion

Todos los endpoints requieren autenticacion (API Key o JWT). La API Key debe tener permisos de acceso a la entidad cuyas cesiones quieres consultar o crear.

## Referencia rapida

| Accion | Metodo | Endpoint |
|--------|--------|----------|
| Listar cesiones | GET | `/cessions?master_entity_id={id}` |
| Detalle de cesion | GET | `/cessions/{id}` |
| Crear cesiones en lote | POST | `/cessions/batch` |
