---
title: 'Preview de documento programado'
openapi: 'GET /v1/scheduled-documents/{id}/preview'
---

Genera un preview en PDF del próximo documento que se emitirá para un documento programado ya guardado. Útil para visualizar cómo se verá el documento antes de su emisión automática.

### Notas

- El PDF generado es un borrador que muestra cómo se verá el documento en su próxima emisión
- El documento no se emite realmente, solo se genera una vista previa
- El PDF está codificado en base64 para facilitar su uso en aplicaciones web
