## ¿Para qué se usa?

Retorna los totales agregados de cobros para un período de fechas dado. Evita tener que paginar y sumar del lado del cliente.

## Qué hace

- Calcula automáticamente la suma de montos brutos, comisiones de Tu Pana y montos netos.
- Retorna el conteo de transacciones pagadas en el período.
- Soporta filtro por `recipient_id` para obtener el resumen de un destinatario específico.

## Ejemplos de uso

- **Liquidación mensual**: Obtener el resumen del mes para mostrárselo a un profesional o empresa.
- **Dashboard de recaudación**: Calcular las métricas del período actual para tu panel de control.
- **Cierre de período**: Verificar los totales antes de generar reportes o conciliaciones.
